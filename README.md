# Workflow Test Project Public

This repository is used to test a professional Git workflow, including branches, Pull Requests, branch protection, staging flow, production flow, and deployment documentation.

## Branches

- `main`: production-ready code
- `staging`: testing before production
- `develop`: development integration
- `feature/*`: temporary task branches

## Goal

The goal is to test the full workflow before applying it to real company projects.
